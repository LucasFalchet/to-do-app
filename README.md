# 📝 To-Do List

Projeto simples de lista de tarefas feito com HTML, CSS e JavaScript puro.

## 🚀 Funcionalidades

- Adicionar tarefas
- Definir prioridade:
  - Alta 🔴
  - Média 🟡
  - Baixa 🔵
- Marcar tarefa como concluída ✔
- Remover tarefa ✖
- Estilização dinâmica conforme prioridade

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
